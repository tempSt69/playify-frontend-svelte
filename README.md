## Getting Started

Rename `.env.template` to `.env` or simply change your personnal `ATLAS_URI` && `ATLAS_TEST_URI`.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/tempSt69/playify-frontend-sveltekit.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Start the project locally
   ```sh
   npm run dev
   ```
